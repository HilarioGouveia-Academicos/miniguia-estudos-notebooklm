# 📓 Miniguia de estudos NotebookLM: Guia Prático de Prestação de Contas Condominial

Este projeto utiliza o **NotebookLM** como uma ferramenta de IA Generativa voltada aos condôminos. O objetivo é desmistificar conceitos fundamentais, compartilhar boas práticas e sanar dúvidas sobre a Prestação de Contas de Condominio Residencial.

---

## 🔗 Link de Acesso
Todo o conteúdo é baseado exclusivamente em fontes teóricas selecionadas, minimizando riscos de alucinação da IA.
👉 [Acessar o NotebookLM do Projeto](https://notebooklm.google.com/notebook/0c32cd02-713f-4f35-91b3-ff3a6a899ea0)

---

## 🎯 Objetivo do Projeto
Criar um miniguia de estudos inteligente capaz de:
- **Explicar** o que é a Prestação de Contas em Condominios Residenciais.
- **Ensinar** os conceitos de Gestão e Governaça Fianceira.
- **Gerar** resumos, relatórios e materiais didáticos personalizados.
- **Sanar** dúvidas específicas utilizando apenas fontes confiáveis e oficiais.

## 🛠️ Funcionalidades Disponíveis
O Notebook permite gerar diversos tipos de materiais para apoio ao estudo:
* ✅ Resumos conceituais
* 🗺️ Mapas mentais
* 📊 Relatórios explicativos
* 🃏 Cartões didáticos (Flashcards)
* 📝 Testes de conhecimento
* 🎨 Infográficos
* 🎥 Resumos em vídeo e áudio

## 📚 Fontes de Dados
Para garantir a precisão, o modelo utiliza:
1.  Entenda como é feita a prestação de contas do síndico - Organize meu Condomínio.
2.  Prestação de Contas em Condomínios Residenciais: Aspectos Jurídicos, Contábeis e Boas Práticas de Governança.
3.  Prestação de contas digital em condomínios: como funciona? - Habitacional.
4.  Qualquer Condômino Pode Pedir Prestação de Contas? Entenda o Que Diz a Lei e a Jurisprudência | Click Síndico.

## 🚀 Como Utilizar
1.  Acesse o link fornecido na descrição.
2.  Interaja com o modelo utilizando **linguagem natural** (como se estivesse conversando com um professor).
3.  Explore as opções de geração de mídia (vídeos, slides e áudios) para facilitar sua compreensão.
